# 🐯 호안 (Hoan) : 당신의 사진에 어울리는 음악

**호안(Hoan)**은 사진 속 순간의 무드와 감성을 분석하여 그에 가장 잘 어울리는 세 곡의 음악을 큐레이션 해주는 웹 애플리케이션입니다. 번거로운 설치 없이 웹 브라우저에서 바로 사진을 업로드하고, 당신의 시각적인 기록을 청각적인 경험으로 확장해 보세요.

🌐 **[호안(Hoan) 라이브 데모 직접 해보기](https://본인깃허브아이디.github.io/레포지토리이름/)**

## 주요 기능

* **직관적인 드래그 앤 드롭 UI**: 클릭 또는 드래그 앤 드롭으로 간편하게 사진을 업로드할 수 있습니다.
* **스마트 감성 큐레이션**: 업로드된 사진의 색감, 피사체, 날씨, 전반적인 분위기를 정밀하게 분석하여 완벽하게 어울리는 3곡의 트랙을 추천합니다.
* **안전한 API 키 관리**: 사용자 본인의 Gemini API 키를 입력하여 작동하며, 입력된 키는 브라우저의 로컬 스토리지(Local Storage)에만 저장되어 외부로 유출되지 않습니다.

## 기술 스택

* **Frontend**: HTML5, CSS3, Vanilla JavaScript
* **AI/ML**: Google Gemini API (`gemini-2.5-flash` 모델)
* **Deployment**: GitHub Pages
* **Library**: [Marked.js](https://marked.js.org/) (마크다운 파싱용)

## 사용 방법

1. [Google AI Studio](https://aistudio.google.com/)에서 **Gemini API Key**를 발급받습니다.
2. [호안 라이브 데모 사이트]((https://본인깃허브아이디.github.io/레포지토리이름/))에 접속합니다.
3. 우측 상단의 ⚙️(설정) 버튼을 눌러 발급받은 API 키를 입력하고 저장합니다.
4. 원하는 사진을 화면에 드래그하여 올려놓거나 클릭하여 업로드합니다.
5. **[음악 추천받기]** 버튼을 누르고 AI가 분석한 맞춤형 플레이리스트를 확인하세요!

## ⚠️ API 키 보안 안내 (중요)

* 본 서비스는 서버를 거치지 않고 사용자의 브라우저에서 직접 Google Gemini API를 호출하는 구조(Client-side)입니다.
* 사용자가 입력한 API 키는 본인의 기기(브라우저 캐시)에만 안전하게 저장되며, 개발자를 포함한 그 누구에게도 전송되지 않습니다.
